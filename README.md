# 창고
창고에 아이템을 넣을 수 있는 시스템입니다.

## 다운로드
* 플러그인 다운로드는 [릴리즈](https://github.com/CosinePlugin/HQAlphaChest/releases)에서 가능합니다.

## 디스코드
* 모든 문의, 질문, 개발 요청은 [디스코드](https://discord.gg/hUkaca9ZQu)에서 해주시길 바랍니다.

## 버전
* 1.12.2~1.20.2

## 관리자 명령어
* /창고관리

## 유저 명령어
* /창고

## 창고 권한
* hqalphachest.use.1~9

## 기능
* 창고에 아이템을 넣을 수 있습니다.
